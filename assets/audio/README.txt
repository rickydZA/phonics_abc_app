AUDIO FILES FOR PHONICS ABC APP
================================

Place your voice recordings in this folder.

FILE NAMING:
- a.mp3, b.mp3, c.mp3, ... z.mp3

CONTENT FORMAT:
Each audio file should contain your voice saying the phonics chant.
Example for letter A: "A A a a a, a is for apple"

SUPPORTED FORMAT:
- MP3 (recommended)
- WAV
- AAC

TIPS:
- Record in a quiet environment
- Speak clearly and at a steady pace
- Keep each recording between 3-5 seconds
