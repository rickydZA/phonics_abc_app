AUDIO FILES FOR PHONICS ABC APP
================================

Place your voice recordings in this folder.

FILE NAMING:
- a.mp3, b.mp3, c.mp3, ... z.mp3

CONTENT FORMAT:
Each audio file should contain your voice saying the phonics chant.
Example for letter A: "A A a a a, a is for apple"

RECOMMENDED SETTINGS:
- Format: MP3 (best compatibility across web, iOS, Android)
- Bitrate: 128 kbps (clear voice, small files)
- Sample Rate: 44.1 kHz
- Channels: Mono (voice doesn't need stereo, halves file size)
- Duration: 3-5 seconds per letter

RECORDING TIPS:
- Record in a quiet room with no echo (a closet with clothes works well)
- Use phone mic or USB microphone
- Keep 6-12 inches distance from microphone
- Maintain consistent volume and distance for all 26 letters
- Avoid clipping (audio peaking into red)

EXPORT SETTINGS (Audacity):
1. File > Export as MP3
2. Bitrate Mode: Constant, 128 kbps
3. Channel Mode: Mono

FREE RECORDING TOOLS:
- Desktop: Audacity (audacityteam.org)
- iOS: Voice Memos
- Android: Recorder

ESTIMATED TOTAL SIZE:
26 letters x ~50KB each = ~1.3 MB
